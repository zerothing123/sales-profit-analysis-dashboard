Sales & Profitability Analysis (Power BI + SQL)
Overview
This project analyzes sales performance, profitability, and discount impact using the Superstore dataset.
The data was first explored and transformed using SQL, and then visualized in Power BI to generate business insights.

Tools Used
SQL (Data extraction & transformation)
Power BI (Dashboard & visualization)

Workflow
1.Used SQL to clean and analyze raw sales data
2.Created aggregated tables (sales, profit, discount analysis)
3.Imported processed data into Power BI
4.Built interactive dashboard for business insights

Key Insights
1.Furniture category is generating losses despite strong sales
2.Higher discount levels negatively impact profitability
3.West region drives the highest sales performance
4.Several products consistently generate losses

Files Included
sales dataset.pbix → Power BI dashboard
sales_data.sql → SQL queries used for analysis
